Donate Today - Assignment Project
This is a simple, responsive website designed as part of my initial learning journey in web development. The project focuses on creating a functional and aesthetic landing page for a donation platform.

🎯 Project Goal
The main objective of this project was to practice:

Semantic HTML: Using proper tags for better structure.

CSS Layouts: Mastering Flexbox and Positioning to align elements.

Responsive Design: Ensuring the site looks good on both mobile and desktop screens.

🛠️ Technologies Used
HTML5 (59.5%)

CSS3 (40.5%)

📋 Features
Interactive Donation Buttons: Hover effects and active states for better UX.

Custom Amount Input: A section for users to input their desired donation amount.

Information Cards: Sections highlighting different causes that need support.

Clean Footer: Including social links and contact information.
